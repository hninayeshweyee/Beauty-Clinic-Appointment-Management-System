# Beauty Clinic Appointment Management System

A web-based application developed using PHP and MySQL to streamline the appointment scheduling and client management process for beauty clinics.

## 🌟 Project Overview
This project focuses on handling the backend logic and database management for a fully functioning clinic booking system, using "Material Dashboard 2" for the frontend presentation.

## 🛠 Features
- Appointment Booking: Seamless user flow to manage client schedules.
- Service & Staff Management: Dynamic data retrieval using PHP.
- Admin Dashboard: Secure backend structure to overview clinic performance and manage records (CRUD operations).
- Relational Database: Optimized schema designed to prevent data redundancy.

## 💻 Tech Stack
- Backend: PHP
- Database: MySQL
- Frontend Framework: HTML, CSS, JavaScript (Integrated with Material Dashboard 2 template)
- Tools: XAMPP, phpMyAdmin, Git
